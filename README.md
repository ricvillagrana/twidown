# Twidown

## Installation
Run the following commands:
```
git clone https://github.com/ricvillagrana/twidown.git
cd twidown
bundle install
yarn
foreman start
```

Go to http://localhost:5000/

## Features
- Registration
- Login
- Posting
- Follow people
- Like posts
- Comment on posts
- Receive an email for each new follower
- The admin receives a daily email with stats
